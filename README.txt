REACTRON - VERSION AVEC CLAVIER + COULEURS

Compilation sous Windows :
gcc main.c jeu.c sauvegarde.c console.c -o reactron.exe

Execution dans Git Bash :
./reactron.exe

Touches dans le jeu :
Z : haut
S : bas
Q : gauche
D : droite
ESPACE : selectionner une case puis echanger avec une case adjacente
E : quitter le niveau

Cette version utilise :
- tableaux 2D
- struct
- fonctions
- pointeurs simples
- fichiers .h / .c
- FILE pour sauvegarde
- conio.h pour lire le clavier sans appuyer sur Entree
- windows.h pour les couleurs console

Important :
Cette version est faite pour Windows.
